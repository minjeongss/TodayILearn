## 배운 부분
### req.params: 라우터의 매개변수
Router에서 :contentId로 넘겨주면, Controller에서 req.params으로 받을 수 있다.
- Router part
```
app.get('/app/now/contents/:contentId',now.showDetailContents);
```
- Controller Part
```
const contentId=req.params.contentId;
```
